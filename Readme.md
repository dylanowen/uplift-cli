# Uplift CLI

### Sit
```bash
uplift sit
```
```bash
uplift sit save
```

### Stand
```bash
uplift stand
```
```bash
uplift stand save
```

### Query
Get the current height
```bash
uplift query
```

### Listen
Continuously get the height
```bash
uplift listen
```

### Set An Arbitrary Height
This doesn't work that well as it has a hard time actually alignign the height correctly
```bash
uplift set 384
```
